# condoku

python based sudoku for your terminal
```
   0  1  2   3  4  5   6  7  8
0|         | 9F    3F|         |
1|         |         |         |
2|    6F   |       5F|         |
 |-----------------------------|
3| 6F    1F|         |         |
4|       7F| 1F      |         |
5| 3F    9F|         |       7F|
 |-----------------------------|
6| 4F      |    7F   |         |
7|         |         |         |
8| 7F      |         |    8F   |

```

## Usage

install python,  
install numpy  
run sudoku.py with python  
```bash
python sudoku.py
```
